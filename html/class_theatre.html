◆ book()

BookResult Theatre::book ( int movie_id,
vector< int > seats )
inline

book() method books the seats in the hall with the given movie_id and returns the hall_id and is_booked status in BookResult struct. If the hall is not found, it returns -1 as hall_id and false as is_booked status. If the seats are not available, it returns the hall_id and false as is_booked status. If the seats are available, it reserves the seats and books them and returns the hall_id and true as is_booked status in BookResult struct.
